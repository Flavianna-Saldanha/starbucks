# ☕ Starbucks Landing Page

Este projeto é uma **landing page inspirada no site da Starbucks**, desenvolvida com foco em **layout moderno, responsivo e boas práticas de front-end**.

O objetivo do projeto foi recriar seções visuais do site utilizando **React, Next.js e TailwindCSS**, aplicando conceitos de **componentização, responsividade e design moderno**.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **Next.js**
* **React**
* **Tailwind CSS**
* **TypeScript**

Essas ferramentas permitem criar interfaces modernas, rápidas e escaláveis.

---

## 🎯 Objetivo do Projeto

O principal objetivo deste projeto foi:

* Praticar **desenvolvimento front-end moderno**
* Trabalhar com **componentização no React**
* Utilizar **TailwindCSS para estilização**
* Criar um **layout responsivo**
* Simular uma **interface inspirada em grandes marcas**

---

## 📱 Responsividade

O layout foi desenvolvido para funcionar em diferentes tamanhos de tela:

* 📱 **Mobile**
* 💻 **Tablet**
* 🖥 **Desktop**

As imagens e seções se reorganizam automaticamente utilizando **Grid e Flexbox do Tailwind**.
